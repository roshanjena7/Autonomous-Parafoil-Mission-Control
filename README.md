# Autonomous Parafoil Mission Control System

An autonomous parafoil navigation simulator developed using Reinforcement Learning and Python.

## Overview

This project simulates an autonomous parafoil payload recovery mission. The system navigates toward a target landing zone while handling wind disturbances and mission constraints.

## Features

* Reinforcement Learning Navigation
* Autonomous Target Tracking
* Terrain Visualization
* Wind Field Simulation
* Landing Zone Guidance
* No-Fly Zone Detection
* Real-Time Telemetry Dashboard
* Flight Path Visualization
* Altitude Monitoring
* Heading Error Analysis
* Mission Scoring System

## Technology Stack

* Python
* NumPy
* Matplotlib
* Reinforcement Learning
